Decisive Analytics Intership

During the summer of 2017, I worked as a UI/UX intern at Decisive Analytics Corporation in Arlington, Virginia. As an intern, I worked on a handul of frontend services used for data analytics and machine learning related applications.

Below are details and screenshots from two projects I worked on. I had the oppurtunity to build both of these UI's' from the ground up and both were passed on to full-time empoyess further development.

SOL Version Control Frontend

This project served at a frontend UI for a version control system (akin to GIT) deisgned specifically for large numbers of structured data objects (think database enteries). The app functions much like a basic bitbucket or gitlab: users can view repositories and compare commits much like one would do with code. Built using Angular and Semantic UI and interfaced with an accociated REST API.



ATLAS UI

Project ATLAS was a project for the U.S. army. The basic idea was that the army was building a video game to simulate and test new eqipment and techniques. Decisive Analytics was contracted to provide data analytics. This UI served as a propotype data visulazation platoform for the project. Since the game istelf was still under development, data for the game DOTA was used as for proof of concept.

A major componet of the project was an alogrhtim that anaylzed player behavviors and dicrete actions.
